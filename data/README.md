# NBA League Averages Data (2000–2024)

**Source:** Basketball-Reference, NBA League Averages — Per Game Stats
**URL:** https://www.basketball-reference.com/leagues/NBA_stats_per_game.html
**Downloaded:** May 2026
**Rows:** 25 seasons (1999-00 through 2023-24)
**Columns used:** Season, Year, PTS (points per game), FGA, 2PA (2-point attempts), 3PA (3-point attempts), 3P% (3-point shooting %)

## Pre-processing
No transformations were applied — the visualization uses the raw league-average values directly. The data is hard-coded as a JavaScript array in `sketches/sketch15-hwk5.js` (lines 33–57) for browser performance. The CSV in this folder is the canonical source.
